# Shopify India Pincode Validator (COD Filter)

A simple tool to block or allow Cash on Delivery (COD) based on Indian pincodes.

## Features
- Validate serviceable pincodes
- Block high-risk regions
- Easy to integrate with Shopify COD forms

## Use Case
Helps Shopify store owners reduce RTO by filtering non-serviceable or risky pincodes.
